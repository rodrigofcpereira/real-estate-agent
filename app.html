📋 Como configurar o Servidor (Google Cloud)

⭐ Recomendado: Google Cloud dá $300 grátis por 90 dias + VM e2-micro grátis para sempre depois.
  1. Acesse cloud.google.com → clique em "Comece gratuitamente"
  2. Faça login com sua conta Google (Gmail já serve)
  3. Vá em Compute Engine → VM Instances → Create Instance
  4. Configure a VM:
    • Machine type: e2-standard-2 (2 CPU, 8GB RAM)
    • Boot disk: Debian 12 ou Ubuntu 22.04
    • Firewall: marque ✅ "Allow HTTP traffic"
  5. Clique em "SSH" para abrir o terminal do servidor
  6. Cole e rode o script de instalação:
curl -fsSL https://raw.githubusercontent.com/rodrigofcpereira/real-estate-agent/main/setup-gcloud.sh | bash
  1. Anote o External IP da VM (ex: 34.95.100.50)
  2. Abra a porta 3000: VPC Network → Firewall → Create Rule → TCP 3000
  3. Volte aqui, selecione Oracle Cloud e cole: http://34.95.100.50:3000
💡 Dica: Após escanear o QR Code no servidor, o WhatsApp fica conectado 24/7 — mesmo com o computador desligado.
